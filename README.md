# Frontend

프론트엔드에서 자주 쓰는 기능들을 모아보자는 생각으로 시작하여,
- Client - Server 모델의 전체적인 흐름을 이해하고
- 각 패키지에 대한 필요성을 체감하며
- 기능 단위로 구현해봄으로써 여러 스킬을 익히는게 주요 목적입니다.

# Tech Stack
- Node v19.0.0
- React v18.2.0
